Сетка арматурная

Сетка арматурная- сварная сетка, которая сваривается из арматуры с распространёнными ячейками: 80х80мм., 100х100мм., 150х150мм., 200х200мм.,250х250мм. Широкое применение сетка приобрела в монолитном строительстве, на строительных объектах где использовалась арматура и вязальная проволока для создания каркаса, теперь для ускорения темпов строительства используется сетка в картах 2000х6000 мм.

Сетка арматурная - лучшая армировка при строительстве. Армирование монолитных и сборных конструкций происходит с помощью сварной арматурной сетки. Она производится из арматуры класса А-ІІІ по ГОСТу 23279-85. Наиболее часто, мы производим арматурные сетки из проволоки от 5 до 18 мм. Сетка, которая производится из такой арматурной проволоки, поставляется в картах.
